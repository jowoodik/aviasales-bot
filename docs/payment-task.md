Задача
Переделеать текущий метод оплаты через сбп или qr кодом на метод оплаты через yookassa.
В .env.example дежит токен для подключения. 

задача по шагам:
1. Убрать текущую реализацию в [subscriptionHandlers.js](../handlers/subscriptionHandlers.js). Сейчас там выводится просто мой номер телефона для оплаты через сбп и мой qr код
2. Создай базу данных для хранения информации об оплате
2. Подкллючить оплату телеграмма с токеном из переменной окружения PAYMENT_TOKEN
3. протестировать что всё работает как надо
4. Описать документацию в файл payment-yookassa.md

Важные моменты:
- Всегда проверяйте получение successful_payment перед выдачей услуги — простой ответ на pre_checkout_query не гарантирует успешную оплату
- Используйте уникальный payload для каждого счета, чтобы идентифицировать транзакцию
- Сохраняйте provider_payment_charge_id из successful_payment для сверки с ЮKassa при необходимости
